# 脱衣ブロック崩し「爆裂ブロック」通常版

[![タイトル画像](img/title.jpg)](https://bakuretuken.github.io/bakuretu-block/)

1999年に作成した Java Applet版「脱衣ブロック崩し」を JavaScript でリメイクした、シンプルな脱衣ブロック崩しゲームです。

**「通常版（1段階ゲーム）」は「重ね着バージョン（2段階ゲーム）」に統合しました。**<br />
「重ね着バージョン」の設定で、「通常版」のゲームも作成可能です。<br />

下記のページをご覧ください。

# [https://github.com/BakuretuKen/Bakuretu-Block2](https://github.com/BakuretuKen/Bakuretu-Block2)
